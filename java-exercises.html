Variabel & Tipe Data

Latihan 1: Deklarasi dan Inisialisasi Variabel

Mudah

Buatlah program Java yang mendeklarasikan dan menginisialisasi variabel dengan tipe data berikut:

  • int (bilangan bulat)
  • double (bilangan desimal)
  • char (karakter)
  • boolean (nilai boolean)
  • String (teks)

Kemudian tampilkan nilai dari masing-masing variabel tersebut.

VariableDeclaration.java 
public class VariableDeclaration {
    public static void main(String[] args) {
        // Deklarasi dan inisialisasi variabel
        int age = 25;
        double height = 175.5;
        char grade = 'A';
        boolean isStudent = true;
        String name = "John Doe";
        
        // Menampilkan nilai variabel
        System.out.println("Nilai variabel int: " + age);
        System.out.println("Nilai variabel double: " + height);
        System.out.println("Nilai variabel char: " + grade);
        System.out.println("Nilai variabel boolean: " + isStudent);
        System.out.println("Nilai variabel String: " + name);
    }
}

Latihan 2: Konversi Tipe Data

Mudah

Buatlah program Java yang melakukan konversi antar tipe data berikut:

  • Konversi dari int ke double
  • Konversi dari double ke int
  • Konversi dari String ke int
  • Konversi dari int ke String
  • Konversi dari char ke int (nilai ASCII)

Tampilkan hasil konversi untuk setiap kasus.

TypeConversion.java 
public class TypeConversion {
    public static void main(String[] args) {
        // Konversi dari int ke double (implicit conversion)
        int num1 = 100;
        double double1 = num1;  // Automatic casting
        System.out.println("int ke double: " + num1 + " -> " + double1);
        
        // Konversi dari double ke int (explicit conversion)
        double double2 = 123.456;
        int num2 = (int) double2;  // Manual casting, decimal part is truncated
        System.out.println("double ke int: " + double2 + " -> " + num2);
        
        // Konversi dari String ke int
        String str1 = "500";
        int num3 = Integer.parseInt(str1);
        System.out.println("String ke int: " + str1 + " -> " + num3);
        
        // Konversi dari int ke String
        int num4 = 250;
        String str2 = Integer.toString(num4);  // or String.valueOf(num4)
        System.out.println("int ke String: " + num4 + " -> " + str2);
        
        // Konversi dari char ke int (nilai ASCII)
        char ch = 'A';
        int ascii = (int) ch;
        System.out.println("char ke int (ASCII): " + ch + " -> " + ascii);
    }
}

Operator & Ekspresi

Latihan 3: Kalkulator Sederhana

Mudah

Buatlah program kalkulator sederhana yang melakukan operasi aritmatika dasar (penjumlahan, pengurangan, perkalian, dan pembagian) pada dua bilangan.

Program harus menerima input dua bilangan dan operator dari pengguna, kemudian menampilkan hasil perhitungan.

SimpleCalculator.java 
import java.util.Scanner;

public class SimpleCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Kalkulator Sederhana");
        System.out.println("--------------------");
        
        // Input bilangan pertama
        System.out.print("Masukkan bilangan pertama: ");
        double num1 = scanner.nextDouble();
        
        // Input operator
        System.out.print("Masukkan operator (+, -, *, /): ");
        char operator = scanner.next().charAt(0);
        
        // Input bilangan kedua
        System.out.print("Masukkan bilangan kedua: ");
        double num2 = scanner.nextDouble();
        
        // Melakukan perhitungan
        double result = 0;
        boolean validOperator = true;
        
        switch (operator) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    result = num1 / num2;
                } else {
                    System.out.println("Error: Pembagian dengan nol tidak diperbolehkan");
                    validOperator = false;
                }
                break;
            default:
                System.out.println("Error: Operator tidak valid");
                validOperator = false;
        }
        
        // Menampilkan hasil
        if (validOperator) {
            System.out.println(num1 + " " + operator + " " + num2 + " = " + result);
        }
        
        scanner.close();
    }
}

Latihan 4: Menghitung Luas dan Keliling Bangun Datar

Mudah

Buatlah program Java yang menghitung luas dan keliling bangun datar berikut:

  • Persegi
  • Persegi panjang
  • Lingkaran
  • Segitiga (dengan rumus Heron)

Program harus menerima input dimensi bangun datar dari pengguna dan menampilkan hasil perhitungan.

ShapeCalculator.java 
import java.util.Scanner;

public class ShapeCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Kalkulator Bangun Datar");
        System.out.println("----------------------");
        System.out.println("1. Persegi");
        System.out.println("2. Persegi Panjang");
        System.out.println("3. Lingkaran");
        System.out.println("4. Segitiga");
        System.out.print("Pilih bangun datar (1-4): ");
        
        int choice = scanner.nextInt();
        
        switch (choice) {
            case 1:
                calculateSquare(scanner);
                break;
            case 2:
                calculateRectangle(scanner);
                break;
            case 3:
                calculateCircle(scanner);
                break;
            case 4:
                calculateTriangle(scanner);
                break;
            default:
                System.out.println("Pilihan tidak valid");
        }
        
        scanner.close();
    }
    
    private static void calculateSquare(Scanner scanner) {
        System.out.print("Masukkan panjang sisi: ");
        double side = scanner.nextDouble();
        
        double area = side * side;
        double perimeter = 4 * side;
        
        System.out.println("Luas persegi: " + area);
        System.out.println("Keliling persegi: " + perimeter);
    }
    
    private static void calculateRectangle(Scanner scanner) {
        System.out.print("Masukkan panjang: ");
        double length = scanner.nextDouble();
        
        System.out.print("Masukkan lebar: ");
        double width = scanner.nextDouble();
        
        double area = length * width;
        double perimeter = 2 * (length + width);
        
        System.out.println("Luas persegi panjang: " + area);
        System.out.println("Keliling persegi panjang: " + perimeter);
    }
    
    private static void calculateCircle(Scanner scanner) {
        System.out.print("Masukkan jari-jari: ");
        double radius = scanner.nextDouble();
        
        double area = Math.PI * radius * radius;
        double perimeter = 2 * Math.PI * radius;
        
        System.out.println("Luas lingkaran: " + area);
        System.out.println("Keliling lingkaran: " + perimeter);
    }
    
    private static void calculateTriangle(Scanner scanner) {
        System.out.print("Masukkan panjang sisi a: ");
        double a = scanner.nextDouble();
        
        System.out.print("Masukkan panjang sisi b: ");
        double b = scanner.nextDouble();
        
        System.out.print("Masukkan panjang sisi c: ");
        double c = scanner.nextDouble();
        
        // Menghitung keliling
        double perimeter = a + b + c;
        
        // Menghitung luas dengan rumus Heron
        double s = perimeter / 2;
        double area = Math.sqrt(s * (s - a) * (s - b) * (s - c));
        
        System.out.println("Luas segitiga: " + area);
        System.out.println("Keliling segitiga: " + perimeter);
    }
}

Latihan 5: Menentukan Kategori Nilai

Menengah

Buatlah program Java yang menerima input nilai ujian (0-100) dan menentukan kategori nilai berdasarkan ketentuan berikut:

  • Nilai 90-100: A
  • Nilai 80-89: B
  • Nilai 70-79: C
  • Nilai 60-69: D
  • Nilai 0-59: E

Program juga harus melakukan validasi input, memastikan nilai yang dimasukkan berada dalam rentang 0-100.

GradeCalculator.java 
import java.util.Scanner;

public class GradeCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Program Penentuan Kategori Nilai");
        System.out.println("-------------------------------");
        
        System.out.print("Masukkan nilai ujian (0-100): ");
        int score = scanner.nextInt();
        
        // Validasi input
        if (score < 0 || score > 100) {
            System.out.println("Error: Nilai harus berada dalam rentang 0-100");
        } else {
            // Menentukan kategori nilai
            char grade;
            
            if (score >= 90) {
                grade = 'A';
            } else if (score >= 80) {
                grade = 'B';
            } else if (score >= 70) {
                grade = 'C';
            } else if (score >= 60) {
                grade = 'D';
            } else {
                grade = 'E';
            }
            
            System.out.println("Nilai " + score + " termasuk kategori: " + grade);
            
            // Menampilkan keterangan tambahan
            switch (grade) {
                case 'A':
                    System.out.println("Keterangan: Sangat Baik");
                    break;
                case 'B':
                    System.out.println("Keterangan: Baik");
                    break;
                case 'C':
                    System.out.println("Keterangan: Cukup");
                    break;
                case 'D':
                    System.out.println("Keterangan: Kurang");
                    break;
                case 'E':
                    System.out.println("Keterangan: Sangat Kurang");
                    break;
            }
        }
        
        scanner.close();
    }
}

Latihan 6: Kalkulator BMI

Menengah

Buatlah program Java yang menghitung Body Mass Index (BMI) dan menentukan kategori berat badan berdasarkan ketentuan berikut:

  • BMI < 18.5: Kekurangan berat badan
  • 18.5 ≤ BMI < 25.0: Normal
  • 25.0 ≤ BMI < 30.0: Kelebihan berat badan
  • BMI ≥ 30.0: Obesitas

Rumus BMI = berat (kg) / (tinggi (m))²

Program harus menerima input berat badan (kg) dan tinggi badan (cm) dari pengguna.

BMICalculator.java 
import java.util.Scanner;

public class BMICalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Kalkulator BMI (Body Mass Index)");
        System.out.println("-------------------------------");
        
        // Input berat badan
        System.out.print("Masukkan berat badan (kg): ");
        double weight = scanner.nextDouble();
        
        // Input tinggi badan
        System.out.print("Masukkan tinggi badan (cm): ");
        double height = scanner.nextDouble();
        
        // Konversi tinggi dari cm ke m
        height = height / 100;
        
        // Validasi input
        if (weight <= 0 || height <= 0) {
            System.out.println("Error: Berat dan tinggi harus lebih dari 0");
        } else {
            // Menghitung BMI
            double bmi = weight / (height * height);
            
            // Menentukan kategori BMI
            String category;
            
            if (bmi < 18.5) {
                category = "Kekurangan berat badan";
            } else if (bmi < 25.0) {
                category = "Normal";
            } else if (bmi < 30.0) {
                category = "Kelebihan berat badan";
            } else {
                category = "Obesitas";
            }
            
            // Menampilkan hasil
            System.out.printf("BMI Anda: %.2f\n", bmi);
            System.out.println("Kategori: " + category);
            
            // Rekomendasi
            System.out.println("\nRekomendasi:");
            if (bmi < 18.5) {
                System.out.println("- Tingkatkan asupan kalori harian");
                System.out.println("- Konsumsi makanan bergizi tinggi");
                System.out.println("- Lakukan latihan kekuatan untuk membangun massa otot");
            } else if (bmi < 25.0) {
                System.out.println("- Pertahankan pola makan dan aktivitas fisik yang seimbang");
                System.out.println("- Lakukan olahraga secara teratur");
            } else {
                System.out.println("- Kurangi asupan kalori harian");
                System.out.println("- Tingkatkan aktivitas fisik");
                System.out.println("- Konsumsi lebih banyak buah dan sayuran");
            }
        }
        
        scanner.close();
    }
}

Latihan 7: Menghitung Faktorial

Menengah

Buatlah program Java yang menghitung faktorial dari sebuah bilangan. Faktorial dari n (dilambangkan dengan n!) adalah hasil perkalian semua bilangan bulat positif dari 1 hingga n.

Contoh: 5! = 5 × 4 × 3 × 2 × 1 = 120

Implementasikan dengan dua cara:

  1. Menggunakan loop (for atau while)
  2. Menggunakan rekursi
FactorialCalculator.java 
import java.util.Scanner;

public class FactorialCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Program Menghitung Faktorial");
        System.out.println("---------------------------");
        
        System.out.print("Masukkan bilangan: ");
        int number = scanner.nextInt();
        
        // Validasi input
        if (number < 0) {
            System.out.println("Error: Faktorial tidak didefinisikan untuk bilangan negatif");
        } else {
            // Menghitung faktorial dengan loop
            long factorialLoop = calculateFactorialLoop(number);
            System.out.println("Faktorial (dengan loop): " + number + "! = " + factorialLoop);
            
            // Menghitung faktorial dengan rekursi
            long factorialRecursion = calculateFactorialRecursion(number);
            System.out.println("Faktorial (dengan rekursi): " + number + "! = " + factorialRecursion);
        }
        
        scanner.close();
    }
    
    // Menghitung faktorial dengan loop
    private static long calculateFactorialLoop(int n) {
        long result = 1;
        
        for (int i = 1; i <= n; i++) {
            result *= i;
        }
        
        return result;
    }
    
    // Menghitung faktorial dengan rekursi
    private static long calculateFactorialRecursion(int n) {
        // Base case
        if (n == 0 || n == 1) {
            return 1;
        }
        
        // Recursive case
        return n * calculateFactorialRecursion(n - 1);
    }
}

Latihan 8: Pola Bintang

Menengah

Buatlah program Java yang mencetak pola bintang berikut berdasarkan input tinggi dari pengguna:

  1. Pola segitiga siku-siku
  2. Pola segitiga sama kaki
  3. Pola berlian (diamond)

Contoh pola segitiga siku-siku dengan tinggi 5:

*
**
***
****
*****
StarPattern.java 
import java.util.Scanner;

public class StarPattern {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Program Pola Bintang");
        System.out.println("-------------------");
        
        System.out.print("Masukkan tinggi pola: ");
        int height = scanner.nextInt();
        
        // Validasi input
        if (height <= 0) {
            System.out.println("Error: Tinggi harus lebih dari 0");
        } else {
            System.out.println("\n1. Pola Segitiga Siku-siku:");
            printRightTriangle(height);
            
            System.out.println("\n2. Pola Segitiga Sama Kaki:");
            printIsoscelesTriangle(height);
            
            System.out.println("\n3. Pola Berlian (Diamond):");
            printDiamond(height);
        }
        
        scanner.close();
    }
    
    // Mencetak pola segitiga siku-siku
    private static void printRightTriangle(int height) {
        for (int i = 1; i <= height; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
    
    // Mencetak pola segitiga sama kaki
    private static void printIsoscelesTriangle(int height) {
        for (int i = 1; i <= height; i++) {
            // Mencetak spasi
            for (int j = 1; j <= height - i; j++) {
                System.out.print(" ");
            }
            
            // Mencetak bintang
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            
            System.out.println();
        }
    }
    
    // Mencetak pola berlian (diamond)
    private static void printDiamond(int height) {
        // Pastikan height ganjil untuk diamond yang sempurna
        if (height % 2 == 0) {
            height++;  // Jika genap, tambahkan 1 agar menjadi ganjil
        }
        
        int middle = (height + 1) / 2;
        
        // Bagian atas diamond (termasuk tengah)
        for (int i = 1; i <= middle; i++) {
            // Mencetak spasi
            for (int j = 1; j <= middle - i; j++) {
                System.out.print(" ");
            }
            
            // Mencetak bintang
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            
            System.out.println();
        }
        
        // Bagian bawah diamond
        for (int i = middle - 1; i >= 1; i--) {
            // Mencetak spasi
            for (int j = 1; j <= middle - i; j++) {
                System.out.print(" ");
            }
            
            // Mencetak bintang
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            
            System.out.println();
        }
    }
}

Latihan 9: Operasi Array

Sulit

Buatlah program Java yang melakukan operasi-operasi berikut pada array:

  1. Mencari nilai maksimum dan minimum
  2. Menghitung rata-rata
  3. Mengurutkan array (ascending dan descending)
  4. Mencari elemen tertentu (linear search dan binary search)
  5. Menggabungkan dua array

Program harus menerima input array dari pengguna.

ArrayOperations.java 
import java.util.Arrays;
import java.util.Scanner;

public class ArrayOperations {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("Program Operasi Array");
        System.out.println("--------------------");
        
        // Input array pertama
        System.out.print("Masukkan jumlah elemen array pertama: ");
        int size1 = scanner.nextInt();
        
        int[] array1 = new int[size1];
        System.out.println("Masukkan elemen array pertama:");
        for (int i = 0; i < size1; i++) {
            System.out.print("Elemen ke-" + (i + 1) + ": ");
            array1[i] = scanner.nextInt();
        }
        
        // Menampilkan array
        System.out.println("\nArray pertama: " + Arrays.toString(array1));
        
        // 1. Mencari nilai maksimum dan minimum
        int max = findMax(array1);
        int min = findMin(array1);
        System.out.println("\n1. Nilai maksimum: " + max);
        System.out.println("   Nilai minimum: " + min);
        
        // 2. Menghitung rata-rata
        double average = calculateAverage(array1);
        System.out.println("\n2. Rata-rata: " + average);
        
        // 3. Mengurutkan array
        int[] ascendingArray = array1.clone();
        sortAscending(ascendingArray);
        System.out.println("\n3. Array setelah diurutkan (ascending): " + Arrays.toString(ascendingArray));
        
        int[] descendingArray = array1.clone();
        sortDescending(descendingArray);
        System.out.println("   Array setelah diurutkan (descending): " + Arrays.toString(descendingArray));
        
        // 4. Mencari elemen tertentu
        System.out.print("\n4. Masukkan elemen yang ingin dicari: ");
        int searchElement = scanner.nextInt();
        
        // Linear search
        int linearIndex = linearSearch(array1, searchElement);
        if (linearIndex != -1) {
            System.out.println("   Linear Search: Elemen " + searchElement + " ditemukan pada indeks " + linearIndex);
        } else {
            System.out.println("   Linear Search: Elemen " + searchElement + " tidak ditemukan");
        }
        
        // Binary search (perlu array terurut)
        int binaryIndex = binarySearch(ascendingArray, searchElement);
        if (binaryIndex != -1) {
            System.out.println("   Binary Search: Elemen " + searchElement + " ditemukan pada indeks " + binaryIndex);
        } else {
            System.out.println("   Binary Search: Elemen " + searchElement + " tidak ditemukan");
        }
        
        // 5. Menggabungkan dua array
        System.out.print("\nMasukkan jumlah elemen array kedua: ");
        int size2 = scanner.nextInt();
        
        int[] array2 = new int[size2];
        System.out.println("Masukkan elemen array kedua:");
        for (int i = 0; i < size2; i++) {
            System.out.print("Elemen ke-" + (i + 1) + ": ");
            array2[i] = scanner.nextInt();
        }
        
        System.out.println("\nArray kedua: " + Arrays.toString(array2));
        
        int[] mergedArray = mergeArrays(array1, array2);
        System.out.println("\n5. Array setelah digabungkan: " + Arrays.toString(mergedArray));
        
        scanner.close();
    }
    
    // Mencari nilai maksimum
    private static int findMax(int[] array) {
        int max = array[0];
        for (int i = 1; i < array.length; i++) {
            if (array[i] > max) {
                max = array[i];
            }
        }
        return max;
    }
    
    // Mencari nilai minimum
    private static int findMin(int[] array) {
        int min = array[0];
        for (int i = 1; i < array.length; i++) {
            if (array[i] < min) {
                min = array[i];
            }
        }
        return min;
    }
    
    // Menghitung rata-rata
    private static double calculateAverage(int[] array) {
        int sum = 0;
        for (int value : array) {
            sum += value;
        }
        return (double) sum / array.length;
    }
    
    // Mengurutkan array (ascending)
    private static void sortAscending(int[] array) {
        Arrays.sort(array);
    }
    
    // Mengurutkan array (descending)
    private static void sortDescending(int[] array) {
        // Pertama urutkan ascending
        Arrays.sort(array);
        
        // Kemudian balik urutan
        for (int i = 0; i < array.length / 2; i++) {
            int temp = array[i];
            array[i] = array[array.length - 1 - i];
            array[array.length - 1 - i] = temp;
        }
    }
    
    // Linear search
    private static int linearSearch(int[] array, int element) {
        for (int i = 0; i < array.length; i++) {
            if (array[i] == element) {
                return i;
            }
        }
        return -1;  // Elemen tidak ditemukan
    }
    
    // Binary search (array harus terurut)
    private static int binarySearch(int[] array, int element) {
        int left = 0;
        int right = array.length - 1;

        int left = 0;
        int right = array.length - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            
            // Jika elemen ditemukan di tengah
            if (array[mid] == element) {
                return mid;
            }
            
            // Jika elemen lebih besar, cari di subarray kanan
            if (array[mid] < element) {
                left = mid + 1;
            }
            // Jika elemen lebih kecil, cari di subarray kiri
            else {
                right = mid - 1;
            }
        }
        
        return -1;  // Elemen tidak ditemukan
    }
    
    // Menggabungkan dua array
    private static int[] mergeArrays(int[] array1, int[] array2) {
        int[] result = new int[array1.length + array2.length];
        
        // Menyalin elemen array pertama
        System.arraycopy(array1, 0, result, 0, array1.length);

        return result;
    }

import java.util.ArrayList;

public class ManajemenMahasiswa {
    private ArrayList daftarMahasiswa;
    private Scanner scanner;
    
    public ManajemenMahasiswa() {
        daftarMahasiswa = new ArrayList<>();
        scanner = new Scanner(System.in);
    }
    
    public void tambahMahasiswa() {
        System.out.println("\n=== Tambah Data Mahasiswa ===");
        
        System.out.print("NIM: ");
        String nim = scanner.nextLine();
        
        // Cek apakah NIM sudah ada
        if (cariMahasiswaByNIM(nim) != null) {
            System.out.println("Error: Mahasiswa dengan NIM " + nim + " sudah ada");
            return;
        }
        
        System.out.print("Nama: ");
        String nama = scanner.nextLine();
        
        System.out.print("Jurusan: ");
        String jurusan = scanner.nextLine();
        
        System.out.print("IPK: ");
        double ipk = Double.parseDouble(scanner.nextLine());
        
        // Validasi IPK
        if (ipk < 0 || ipk > 4.0) {
            System.out.println("Error: IPK harus dalam rentang 0-4.0");
            return;
        }
        
        Mahasiswa mahasiswa = new Mahasiswa(nim, nama, jurusan, ipk);
        daftarMahasiswa.add(mahasiswa);
        
        System.out.println("Data mahasiswa berhasil ditambahkan");
    }
    
    public void tampilkanSemuaMahasiswa() {
        System.out.println("\n=== Daftar Mahasiswa ===");
        
        if (daftarMahasiswa.isEmpty()) {
            System.out.println("Belum ada data mahasiswa");
            return;
        }
        
        for (int i = 0; i < daftarMahasiswa.size(); i++) {
            System.out.println((i + 1) + ". " + daftarMahasiswa.get(i));
        }
    }
    
    public Mahasiswa cariMahasiswaByNIM(String nim) {
        for (Mahasiswa mahasiswa : daftarMahasiswa) {
            if (mahasiswa.getNim().equals(nim)) {
                return mahasiswa;
            }
        }
        return null;
    }
    
    public void cariMahasiswa() {
        System.out.println("\n=== Cari Mahasiswa ===");
        
        System.out.print("Masukkan NIM: ");
        String nim = scanner.nextLine();
        
        Mahasiswa mahasiswa = cariMahasiswaByNIM(nim);
        
        if (mahasiswa != null) {
            System.out.println("Mahasiswa ditemukan:");
            System.out.println(mahasiswa);
        } else {
            System.out.println("Mahasiswa dengan NIM " + nim + " tidak ditemukan");
        }
    }
    
    public void hapusMahasiswa() {
        System.out.println("\n=== Hapus Data Mahasiswa ===");
        
        System.out.print("Masukkan NIM: ");
        String nim = scanner.nextLine();
        
        Mahasiswa mahasiswa = cariMahasiswaByNIM(nim);
        
        if (mahasiswa != null) {
            daftarMahasiswa.remove(mahasiswa);
            System.out.println("Data mahasiswa berhasil dihapus");
        } else {
            System.out.println("Mahasiswa dengan NIM " + nim + " tidak ditemukan");
        }
    }
    
    public void urutkanByIPK() {
        System.out.println("\n=== Urutkan Mahasiswa Berdasarkan IPK (Descending) ===");

            System.out.println("Belum ada data mahasiswa");
            return;
        }

        Collections.sort(daftarMahasiswa, new Comparator() {
            @Override

import java.util.ArrayList;
import java.util.Scanner;
import java.util.UUID;

public class Perpustakaan {
    private ArrayList daftarBuku;
    private ArrayList daftarPeminjaman;
    private Scanner scanner;
    
    public Perpustakaan() {
        daftarBuku = new ArrayList<>();
        daftarPeminjaman = new ArrayList<>();
        scanner = new Scanner(System.in);
        
        // Menambahkan beberapa buku contoh
        tambahBukuContoh();
    }
    
    private void tambahBukuContoh() {
        daftarBuku.add(new Buku("B001", "Java Programming", "John Doe", 2020, 5));
        daftarBuku.add(new Buku("B002", "Python Basics", "Jane Smith", 2019, 3));
        daftarBuku.add(new Buku("B003", "Data Structures and Algorithms", "Robert Johnson", 2018, 2));
    }
    
    public void tambahBuku() {
        System.out.println("\n=== Tambah Buku ===");
        
        String id = "B" + String.format("%03d", daftarBuku.size() + 1);
        
        System.out.print("Judul: ");
        String judul = scanner.nextLine();
        
        System.out.print("Penulis: ");
        String penulis = scanner.nextLine();
        
        System.out.print("Tahun Terbit: ");
        int tahunTerbit = Integer.parseInt(scanner.nextLine());
        
        System.out.print("Jumlah: ");
        int jumlah = Integer.parseInt(scanner.nextLine());
        
        Buku buku = new Buku(id, judul, penulis, tahunTerbit, jumlah);
        daftarBuku.add(buku);
        
        System.out.println("Buku berhasil ditambahkan dengan ID: " + id);
    }
    
    public void tampilkanSemuaBuku() {
        System.out.println("\n=== Daftar Buku ===");
        
        if (daftarBuku.isEmpty()) {
            System.out.println("Belum ada buku");
            return;
        }
        
        for (int i = 0; i < daftarBuku.size(); i++) {
            System.out.println((i + 1) + ". " + daftarBuku.get(i));
        }
    }
    
    public void cariBuku() {
        System.out.println("\n=== Cari Buku ===");
        
        System.out.print("Masukkan judul atau penulis: ");
        String keyword = scanner.nextLine().toLowerCase();
        
        ArrayList hasilPencarian = new ArrayList<>();
        
        for (Buku buku : daftarBuku) {
            if (buku.getJudul().toLowerCase().contains(keyword) || 
                buku.getPenulis().toLowerCase().contains(keyword)) {
                hasilPencarian.add(buku);
            }
        }
        
        if (hasilPencarian.isEmpty()) {
            System.out.println("Buku tidak ditemukan");
        } else {
            System.out.println("Hasil pencarian:");
            for (int i = 0; i < hasilPencarian.size(); i++) {
                System.out.println((i + 1) + ". " + hasilPencarian.get(i));
            }
        }
    }
    
    public void pinjamBuku() {
        System.out.println("\n=== Pinjam Buku ===");
        
        tampilkanSemuaBuku();
        
        System.out.print("Masukkan ID buku yang ingin dipinjam: ");
        String idBuku = scanner.nextLine();
        
        Buku buku = null;
        for (Buku b : daftarBuku) {
            if (b.getId().equals(idBuku)) {
                buku = b;
                break;
            }
        }
        
        if (buku == null) {
            System.out.println("Buku dengan ID " + idBuku + " tidak ditemukan");
            return;
        }
        
        if (buku.getTersedia() <= 0) {
            System.out.println("Maaf, buku " + buku.getJudul() + " sedang tidak tersedia");
            return;
        }
        
        System.out.print("Nama Peminjam: ");
        String namaPeminjam = scanner.nextLine();
        
        // Generate ID peminjaman
        String idPeminjaman = "P" + UUID.randomUUID().toString().substring(0, 8);
        
        // Buat objek peminjaman
        Peminjaman peminjaman = new Peminjaman(idPeminjaman, buku.getId(), buku.getJudul(), namaPeminjam);
        daftarPeminjaman.add(peminjaman);
        
        // Update status buku
        buku.pinjam();
        
        System.out.println("Buku berhasil dipinjam dengan ID peminjaman: " + idPeminjaman);
    }
    
    public void kembalikanBuku() {
        System.out.println("\n=== Kembalikan Buku ===");
        
        System.out.print("Masukkan ID peminjaman: ");
        String idPeminjaman = scanner.nextLine();
        
        Peminjaman peminjaman = null;
        for (Peminjaman p : daftarPeminjaman) {
            if (p.getId().equals(idPeminjaman)) {
                peminjaman = p;
                break;
            }
        }
        
        if (peminjaman == null) {
            System.out.println("Peminjaman dengan ID " + idPeminjaman + " tidak ditemukan");
            return;
        }
        
        if (peminjaman.isStatusKembali()) {
            System.out.println("Buku sudah dikembalikan sebelumnya");
            return;
        }
        
        // Update status peminjaman
        peminjaman.kembalikan();
        
        // Update status buku
        for (Buku buku : daftarBuku) {
            if (buku.getId().equals(peminjaman.getIdBuku())) {
                buku.kembali();
                break;
            }
        }
        
        System.out.println("Buku berhasil dikembalikan");
    }
    
    public void tampilkanHistoryPeminjaman() {

        if (daftarPeminjaman.isEmpty()) {

            return;
        }
        
        for (int i = 0; i < daftarPeminjaman.size(); i++) {
            System.out.println((i + 1) + ". " + daftarPeminjaman.get(i));
        }
    }
    
    public void tampilkanMenu() {

        System.out.println("2. Tampilkan Semua Buku");

        System.out.println("5. Kembalikan Buku");
        System.out.println("6. Tampilkan History Peminjaman");
        System.out.println("0. Keluar");
        System.out.print("Pilih menu: ");
    }
    
    public void jalankan() {
        int pilihan;
        
        do {
            tampilkanMenu();
            pilihan = Integer.parseInt(scanner.nextLine());
            
            switch (pilihan) {
                case 1:
                    tambahBuku();
                    break;
                case 2:
                    tampilkanSemuaBuku();
                    break;
                case 3:
                    cariBuku();
                    break;

                    break;

                    break;

                    break;
                case 0:

                    break;
                default:
                    System.out.println("Pilihan tidak valid");
            }
        } while (pilihan != 0);
        
        scanner.close();
    }

    }

4. Apa perbedaan utama antara ArrayList dan LinkedList?

  • ArrayList tidak dapat menyimpan tipe data primitif, sedangkan LinkedList bisa
  • ArrayList memiliki ukuran tetap, sedangkan LinkedList memiliki ukuran dinamis
  • ArrayList lebih cepat untuk akses acak, sedangkan LinkedList lebih cepat untuk operasi insert/delete
  • ArrayList hanya dapat menyimpan objek String, sedangkan LinkedList dapat menyimpan semua jenis objek
